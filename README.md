# arionum-miner
This is a multithreaded Python miner for the
[Arionum](https://www.arionum.com/) cryptocurrency.

# Installation

## Ubuntu/Debian
```
sudo apt install python3-pip
sudo pip3 install -r requirements.txt
```

# Usage

```
python3 miner.py --wallet <CHANGEADDRESS>
```
